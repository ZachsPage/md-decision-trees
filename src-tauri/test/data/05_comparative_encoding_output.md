# 05_comparative_encoding_output (md-decision-trees)

D: Decision Node
* O: Option 1
  * P: Pro for Option 1
  * P,1,4: Pro for Options 1 and 2
* O: Option 2
  * C: Con for Option 2
  * P,4-C,1,7: Pro for Option 2, Con for Options 1 and 3
* O: Option 3
  * C,7-P,1,4: Con for Option 3, but Pro for 1 and 2
